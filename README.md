API Testing com Postman - Projeto Prático

Sobre o projeto
Projeto desenvolvido para praticar testes de API REST utilizando Postman, simulando operações reais de CRUD em um sistema de usuários.

Tecnologias
- Postman
- API REST (JSONPlaceholder)

Funcionalidades testadas
- GET /users → Listagem de usuários
- GET /users/1 → Consulta específica
- POST /users → Criação de usuário
- PUT /users/1 → Atualização de dados
- DELETE /users/1 → Remoção de usuário

Evidências
Os testes realizados estão documentados na pasta `/prints`.

Collection
A collection do Postman está disponível na pasta `/collection`.

Objetivo
Demonstrar conhecimento em:
- Consumo de APIs REST
- Testes de endpoints
- Interpretação de status HTTP
- Organização de testes no Postman
